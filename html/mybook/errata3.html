『趣味で量子力学』正誤表

読者からの報告に感謝しつつ。

初版・第2版に共通する間違い
ページ行間違い正しくは・・・
P.404 行目　分母が (n+1) になっている　(n+l) とすべき。
P.859 行目　(-2 sinh(k'L) + …になっている　(-2 の直後で括弧を閉じるべき。
P.106下から 7 行目　d^2ψ/dx^2 　偏微分の記号で ∂^2ψ/∂x^2 と表記すべき。
P.1403 行目　数式に n が含まれている。　n は要らない。
P.2066 行目　∂θ/∂x　∂φ/∂x
P.214一番下　F(x)　F(k)


その他

　p.41 において、方位量子数 l = 0, 1, 2, … に対応する記号 s, p, d, f に関して、それ以降は「アルファベット順に g, h, i, … と続くらしい」という記述がある。 しかし i と j が紛らわしいので i を除外して「g, h, j, k, …」とするという流儀もあるようだ。 使う人はほとんどいないだろうが、そういう文献もどこかにあるらしいということでご参考までに。
