\section{運動量を示すベクトル}


　シュレーディンガー方程式を立てた時のことを思い出してもらいたい。
　波動関数を位置座標で微分して$ -i\hbar $を掛けることで運動量を取り出せるのであった。
　どうやら波動関数には位置についての情報の他に、
運動量についての情報も「同時に」含まれているようである。
　前回は波動関数が無限の座標ベクトルの組み合わせで表現できることを確認したが、
このように表された状態ベクトルの一体どこに運動量の情報が隠されているというのだろう？

<P>

　無限の座標ベクトルと直交する別の無限個のベクトルがあって、
それらの方向が運動量を表しているなどと考えるのは無理がある。
　波動関数は座標ベクトルの組み合わせだけですでに完全に表されているではないか。
　もしそんなベクトルがあれば、それは座標ベクトルと区別がつかないものになるだろう。
　座標ベクトルだって他の全ての座標ベクトルと直交しつつ無限に存在するのだから。

<P>

　では座標ベクトルとは直交しない形で存在する何らかのベクトルが
運動量の情報を指し示していると考えればいいのだろうか。
　とりあえずそれを「\red{運動量ベクトル}」と呼び、$ |p\rangle $と表すことにしよう。
　粒子がある特定の運動量を確実に持つという状態にあれば、
ある一つの運動量ベクトルの方向を向いていることになる。
　当然のことだが、確実にある運動量を持つという状態は別の運動量を持たない状態であるから、
やはりこれらのベクトル$ |p\rangle $どうしもこの空間内で直交して存在していると考えられる。

<P>

　以上の説明で、位置と運動量を同時に決められないという量子力学の論理構造を
イメージとして掴んでもらえたのではないかと思う。
　状態ベクトルが、ある一つの座標ベクトルの方向を向きながら同時に
ある一つの運動量ベクトルの方向を向くことは不可能なのだ。

<P>

　ではそのような運動量ベクトルと座標ベクトルの位置関係は一体どのようなものなのだろうか。
　内積を取れば、0 以上 1 以下の範囲にあることは確かだ。
　0 と 1 は含まないであろう。
　0 なら直交してしまうし、1 なら重なってしまうからだ。
　しかし現段階ではこれ以上のことは分からない。
　別の視点から考え直そう。

\section{指数関数への分解}

　波動関数から運動量を取り出すときに、微分しても関数の形が変わらないことが大切であった。
　関数の形が変わってしまえば、変形後の関数の一体どの部分が
中から飛び出してきた運動量の値を表しているかの判別が付かなくなってしまう。
　そういうことがないように指数関数を使ったのだった。
　指数関数ならば確実にどんな運動量を持つかが調べられる！
　というわけで波動関数を多数の指数関数に分解して考えてみたらどうだろうか。
　何とうまい具合だろう。
　実際これが出来るのだ。
　次のような指数関数の組み合わせは$ -\pi \leqq x \leqq \pi $の範囲内で完全直交系になっているのである。

<tex>
e^{ix}\ ,\ e^{-ix}\ ,\ e^{2ix}\ ,\ e^{-2ix}\ ,\ e^{3ix}\ ,\ e^{-3ix}\ ,\ e^{4ix}\ , \cdots
</tex>

　ただし、このままでは規格化されていないので自分自身との内積を取った時に
ちゃんと 1 になるように係数を付けてやる必要がある。
　例えば、$ e^{ix} $の場合には、

<tex>
\int_{-\pi}^{\pi} ( A e^{ix} )^{\ast} A e^{ix} \diff x \ &=\ \int_{-\pi}^{\pi} A^{\ast} A e^{-ix} e^{ix} \diff x \\
&=\ \int_{-\pi}^{\pi} |A|^{2} \diff x \\
&=\ 2\pi |A|^{2} \ =\ 1 \\
&\!\!\!\!\!\!\!\!\!\!\!\!\!\! \therefore\ A\ =\ \frac{1}{\sqrt{2\pi}}
</tex>

という具合になるが、この計算を見ればどの関数にも同じ係数を付けてやればいいことが分かるだろう。

<tex>
{ \textstyle \frac{1}{\sqrt{2\pi}} } e^{ix}\ ,
{ \textstyle \frac{1}{\sqrt{2\pi}} } e^{-ix}\ ,
{ \textstyle \frac{1}{\sqrt{2\pi}} } e^{2ix}\ ,
{ \textstyle \frac{1}{\sqrt{2\pi}} } e^{-2ix}\ ,
{ \textstyle \frac{1}{\sqrt{2\pi}} } e^{3ix}\ , \cdots
</tex>

　範囲がこれでは使いにくいというのであれば$ x $のスケールをいじることで、
$ -L \leqq x \leqq L $の範囲で完全直交系になるようにも出来る。
　係数も付け直さなければならないが大した作業ではない。

<tex>
{ \textstyle \frac{1}{\sqrt{2L}} } e^{i\frac{\pi}{L} x}\ ,
{ \textstyle \frac{1}{\sqrt{2L}} } e^{-i\frac{\pi}{L} x}\ ,
{ \textstyle \frac{1}{\sqrt{2L}} } e^{2i\frac{\pi}{L} x}\ ,
{ \textstyle \frac{1}{\sqrt{2L}} } e^{-2i\frac{\pi}{L} x}\ , \cdots\ ,
{ \textstyle \frac{1}{\sqrt{2L}} } e^{i\frac{\pi}{L}n x}\ , \cdots
</tex>

　一番左の$ \frac{1}{\sqrt{2L}} e^{i\frac{\pi}{L} x} $は運動量が
$ p = \hbar \pi/L $である状態を表す。
　次は$ p = -\hbar \pi/L $である状態。
　順に、$ p = 2\hbar \pi/L $、$ -2\hbar \pi/L $、$ 3\hbar \pi/L $、
$ -3\hbar \pi/L $、$ 4\hbar \pi/L $、
$ \cdots $、$ n\hbar \pi/L $、$ \cdots $と続く。
　波動関数$ \psi(x) $がこれらの指数関数をどのような割合で含むのかを
計算したければ関数の内積を取ってやればいい。

<tex>
\int_{-L}^{L} ( {\textstyle \frac{1}{\sqrt{2L}}} e^{i\frac{\pi}{L}n x} )^\ast \ \psi(x) \diff x
</tex>

　これはベクトル表現で表せば、

<tex>
\langle {\textstyle \frac{1}{\sqrt{2L}}} e^{ i\frac{\pi}{L}n x} | \psi \rangle 
</tex>

ということであり、こうして求められる係数と、指数関数をベクトル表現したものの組み合わせで
状態ベクトルを表すことが出来るのである。
　どうやら指数関数をベクトルで表したものが$ |p\rangle $の正体であると考えて良さそうだ。

<tex>
| p \rangle \ =\ | {\textstyle \frac{1}{\sqrt{2L}}} e^{ipx/\hbar} \rangle
</tex>

　実際に個々の指数関数どうしは直交してるわけだし、
このベクトルは運動量$ p $を持つ状態を表している。
　まさに冒頭で考えたイメージとぴったりではないか。

<P>

　範囲を制限した波動関数はこのような飛び飛びの運動量を持つ状態の和として表せることが分かった。
　逆に言えば、限られた範囲に粒子を閉じ込めてしまうと、
その運動量はこのような飛び飛びの値しか取れなくなるということを意味している。

<P>

　しかしこの粒子は本当の意味で閉じ込められているわけではない。
　周期的に同じ形の波動関数が無限に繰り返すことが許されているからだ。
　つまり、ここで論じた内容は「<B>周期的境界条件</B>」を適用した場合の状況を表している。
　本当の意味で閉じ込められた粒子にはもう少し厳しい条件が付く。

\section{完全に閉じ込められた粒子}

　完全に閉じ込められた状態というのは、この領域の外側での粒子の存在確率が 0 であるということであり、
波動関数が外側の領域と連続的に繋がるためには$ x = - L $と$ x = L $での波動関数の値が
 0 にならなければならない。

<P>

　なぜ波動関数が連続していなければならないかを聞かれると少し困る。
　シュレーディンガー方程式は微分方程式なので不連続な点があると使えないからだ、
と説明すれば何割かの人はごまかせるだろう。
　しかしシュレーディンガー方程式は 2 階微分を含むので、
微分したものも連続でなければならないはずではないか、と反論されれば何も言えない。
　確かに周期的境界条件を使う時にはちゃんと 2 階微分の連続にまで気を配っているのだが、
「無限の井戸型ポテンシャル」の場合にはそこまではこだわらない。
　まぁ、ある地点を境にいきなりポテンシャルが無限大になるという仮定自体が不自然なものなので
そこは飽くまでも近似ということで大目に見てもらいたいところだ。

<P>

　自然界には一見不連続に見える現象は数多いけれども、
数学的に厳密に不連続になるようなことはなく、
その不連続に見えるぎりぎりの領域を拡大して調べてやると、
実に色んな振る舞いが見られるものである。

\note{　例えば水と空気の境目は不連続のようだが、
そこはくっきり分かれているわけではなく、
水分子と空気分子の激しい運動、蒸発と凝固、さらに細かく見れば
そこには複雑な電子の営みがあって嵐の時の海のようになっているわけだ。<br>
　電子回路のオンとオフには明確な違いがあるが、
その瞬間に起こる過渡現象は実に微妙で連続したものである。<br>
　ある一定の温度、ある一定の濃度で性質が激変する化学物質の構造も、
何の前触れもなしに変化が完了するわけではない。<br>
　もしそういう中間状態のない厳密な不連続があるとすれば
それは「特異点」と呼ばれて非常な議論の対象になるだろう。}

　話を戻そう。
　ある領域内にのみ存在する粒子の波動関数の値が両端で 0 にならなければならないとすると、
許される波の種類はそう多くない。
　$ \sin $関数か$ \cos $関数、あるいはそれらに虚数$ i $を掛けたものが候補となる。
　ちょっと工夫してやれば指数関数の簡単な組み合わせで$ \cos $関数を表現することが
できるのだが、例えば次のような具合だ。

<tex>
\cos\ x \ &=\ \frac{1}{2} \left\{ ( \cos\ x + i\ \sin\ x ) + ( \cos\ x - i\ \sin\ x ) \right\} \\
&=\ \frac{1}{2}\ ( e^{ix} + e^{-ix} )
</tex>

　先に話した条件に合うためには本当は$ \cos x $ではなく、
奇数$ n $を使って$ \cos n(\pi/2L)x $とせねばならないのだが、
そうするとやたら複雑な式変形をしている印象を与えてしまうので、
式変形の本質が分かるように略して書いてみた。

<P>

　これは運動量が$ p = n\hbar(\pi/2L) $を取る状態と
$ p = - n\hbar(\pi/2L) $を取る状態の重ね合わせを意味している。
　つまり粒子が閉じ込められた状態では、運動量が$ p $を取る状態と$ -p $を取る状態が
同じだけの割合で重なっていなければならないのである。
　この結果は面白いほど理にかなっている。
　右へ行く粒子が観測される割合と左へ進む粒子が観測される割合がいつも釣り合っており、
確率の言葉を借りるならば「この状況での運動量の期待値は 0」だということだ。
　長期的に見てどこかへ行ってしまうことはないというイメージであり、
まさに束縛された状態を表している。

<P>

　同様な方法で$ \sin $関数も作ることができるだろう。

<tex>
\sin\ x \ &=\ - \frac{1}{2}\ i\ \left\{ ( \cos\ x + i\ \sin\ x ) - ( \cos\ x - i\ \sin\ x ) \right\} \\
&=\ - \frac{1}{2}\ i\ ( e^{ix} - e^{-ix} ) 
</tex>

　これも$ n $が偶数という違いだけで、許される運動量はさっきと同じ式で表されることになる。
　2 つの指数関数の係数の符号が違うので、
右向きと左向きが同じ割合で重なってはいないのではないかと不審に思うかも知れないが、
実際に観測される確率は係数の絶対値の 2 乗で決まるのだからそんな心配は無用である。

\section{連続的な運動量}

　もし波動関数が周期的ではなく、有限の範囲を越えて無限の彼方まで存在するものだったらどうだろう？
　無限の範囲に横たわる関数を別の周期的な関数で展開してやるためには
連続的な和、すなわち積分が必要なのであった。
　つまりあらゆる連続的な波長を持つ基本波の
重ね合わせによってしか表現できなくなってしまう。
　すなわち運動量も座標のように連続値を取るようになるわけだ。
　その時の係数も連続的に変化する量を考えなくてはならないので、
$ \xi(p) $のような関数を使うことになる。

<tex>
\psi(x)\ =\  \frac{1}{\sqrt{2\pi\hbar}} \int_{-\infty}^{\infty} \xi(p) e^{ipx/\hbar} \diff p
</tex>

　ここに出てくる関数$ \xi(p) $の絶対値の 2 乗は、
粒子の運動量を観測した時にその値が$ p $として見出される確率を表すものであり、
波動関数に良く似たものである。
　それでこの運動量の分布を表す関数$ \xi(p) $を
「\red{運動量表示の波動関数}」と呼ぶ。
　普通の波動関数$ \psi(x) $は「座標表示の波動関数」だというわけだ。
　このように理論構造上は運動量も座標も対等になっている。

<P>

　実際、上の式は「\red{フーリエ逆変換}」と呼ばれるものと同じ形式である。
　運動量表示と座標表示の間にこのような関係が成り立っているということは、
逆に座標表示から運動量表示への変換は次のような
「\red{フーリエ変換}」になっていることが数学の要請から分かる。

<tex>
\xi(p)\ =\ \frac{1}{\sqrt{2\pi\hbar}} \int_{-\infty}^{\infty} \psi(x) e^{-ipx/\hbar} \diff x
</tex>

　フーリエ変換というのは元の関数$ \psi(x) $に
波動成分$ e^{ipx/\hbar} $がどの程度含まれているか、という分布を
成分表示する関数を求めているのであり、工学分野でもスペクトル分析に重宝しているものだ。
　フーリエ変換の指数関数の右肩がマイナスになっているのは、
複素関数の内積を取る時には片方の関数を複素共役にしなくてはならないことによる。
　あとでこの式をベクトル表示で書き直すのでその時に見比べてもらいたい。

<P>

　これらの式の初めに$ 1/\sqrt{2\pi\hbar} $という係数が
必要なのは完全系関数の規格化に関わる要請であるが、詳しい理由については
フーリエ変換を学んでもらうとしよう。

<P>

　数学で習ったフーリエ変換には$ \hbar $なんてものは出てこないので、
その違いが気になる人は、運動量$ p $の代わりに$ k = p/\hbar $と
定義される「\red{波数}」という量を使えば全く同じ形式にすることが出来る。
　この量を波数と呼ぶ理由は$ p = h/\lambda $という関係を思い出してもらえば分かる。
　$ k = 2\pi/\lambda $である。
　すなわち、長さ$ 2\pi $メートル中にある波の数を表しているわけだ。
　このように波数$ k $は数式が簡単になるというので運動量$ p $の代わりに良く使われている。
　しかし私はしばらくはその書き方を使わないだろう。

<P>

　上では$ |p\rangle $は指数関数$ \textstyle \frac{1}{\sqrt{2L}} \displaystyle e^{ipx/\hbar} $をベクトル表現したものであると説明したが、
これは離散的な運動量の場合の話であって、
連続的な場合のベクトル$ |p\rangle $は$ \frac{1}{\sqrt{2\pi\hbar}} e^{ipx/\hbar} $を
ベクトル表現したものであると言えるようだ。

\section{連続と不連続の統合}

　そろそろまとめに取り掛かろう。
　状態ベクトル$ |\psi\rangle $は座標表示のときと同じように
直交する無限の運動量ベクトル$ |p\rangle $の組み合わせで表せるのであり、
運動量が離散値を取る場合には

<tex>
|\psi\rangle\ =\ \sum_{i}^{\infty} \sigma_i\ |p_i\rangle
</tex>

という具合である。
　このときの係数$ \sigma_i $は

<tex>
\sigma_i\ =\ \langle p_i| \psi\rangle
</tex>

と表せる。
　座標表示の場合から類推できると思うが、
こうして求められる係数の絶対値の 2 乗の値が、
この粒子の運動量を観測した時に運動量$ p $を持つ確率を表していることになる。

<tex>
P\ =\ | \langle p_i | \psi \rangle |^2
</tex>

　運動量が連続的な値を取らないので、座標表示の時のような積分が必要ないのが嬉しいではないか。

<P>

　次に運動量が連続値を取る場合についてまとめてみよう。
　形式は離散値を取る場合とあまり変わらない。
　ただ和の記号の代わりに積分を使うことと、
$ \sigma_i $の場合とは違って係数が連続的に変化するので
$ \xi(p) $のように表さなければならないことだけである。

<tex>
|\psi\rangle\ =\ \int_{-\infty}^{\infty} \diff p\ \xi(p)\ |p\rangle
</tex>

　ここで関数$ \xi(p) $は、

<tex>
\xi(p)\ =\ \langle p | \psi\rangle
</tex>

と表せる。
　先ほどのフーリエ変換の式と見比べてみよう。
　あの式はこれを意味していたわけだ。
　粒子の運動量を観測した時に、その運動量が$ p $から$ p + \diff p $の間に見出される確率は、

<tex>
P\ =\ |\xi(p)|^2 \diff p
</tex>

と計算してやればいい。
　座標表示の波動関数と全く同じような形式が成り立っているわけだ。

<P>

　連続的な場合と離散的な場合とでこれだけ形式が似ているというのに扱いが違うのがどうも気持ち悪い。
　実際、これら 2 つの場合は完全に別個に起こるものではない。
　例えば、中途半端な深さの箱型ポテンシャルを考えると、
運動量が小さいうちは粒子は箱の内側に捕らえられてしまい離散的な値を取るが、
運動量が大きくなってくるとポテンシャルに縛られることなく、
自由な粒子として振舞うようになる。
　これらを一つの形式でまとめて理解してやることは出来ないのだろうか？
　実はデルタ関数を使ってちょっと工夫すればそれが出来るのである。

<P>

　状態ベクトルが連続な運動量ベクトルを使って、

<tex>
|\psi\rangle\ =\ \int_{-\infty}^{\infty} \diff p \ \Psi(p)\ |p\rangle
</tex>

のように表されているとしよう。
　連続的な部分と不連続な部分を合わせた波動関数$ \Psi(p) $は、
連続的な部分の関数を$ \xi(p) $とすると、

<tex>
\Psi(p)\ =\ \sum_i \sigma_i \delta(p-p_i) \ +\ \xi(p)
</tex>

と表現してやればいいことになる。
　連続的な波動関数の所々に無限の長さの突起が出ているような関数である。
　もちろんデルタ関数を使っているだけなので、運動量の確率を計算するために
この関数を積分しても値が無限になってしまうことはない。

\section{位置ベクトルとの位置関係}

　ここまで来れば初めの疑問に答えるのは簡単だ。
　座標ベクトル$ |x\rangle $と運動量ベクトル$ |p\rangle $の内積はデルタ関数と指数関数の内積であり、

<tex>
\langle x | p \rangle\ &=\ \frac{1}{\sqrt{2\pi\hbar}} \int_{-\infty}^{\infty} \delta(x'-x)\ e^{i\frac{p}{\hbar}x'} \diff x' \\
&=\ \frac{1}{\sqrt{2\pi\hbar}} e^{i\frac{p}{\hbar}x}
</tex>

のように計算してやればいい。
　ちょっと複雑に見えるかも知れないが、「ある位置$ x $に見出される状態」を表すための座標$ x $と、
積分するための座標$ x' $を区別しなくてはならないので記号がごちゃごちゃして見えるだけだ。

<P>

　ああ、何という見当外れだろう！
　確かに、内積の値は予言したように 1 にも 0 にもならないことが分かる。
　しかしそんなことより値が複素数だという、
いんちきパズルの答えのようなことになっているではないか。
　しかも運動量も座標も連続であるために、このままでは普通の内積のイメージとして扱うことすら出来ない。
　実際、この値の絶対値は 0 と 1 の間に収まるどころか、1 をはるかに越えているではないか。
　座標ベクトルどうしの内積もデルタ関数であって 1 をはるかに越えていたことを思い出してもらいたい。

　これはもう通常のイメージの届かない話だ。
　恐るべし複素ベクトル空間。
　座標ベクトルと運動量ベクトルはこんな抽象的な位置関係になっているのである。

<P>

　しかしここで思考停止する必要はなくて、もう少し解釈してやることは出来る。
　座標$ x $の値をどれかに固定した時、運動量$ p $をどんな値にしようとも、
この内積の値の絶対値は皆同じなのではないだろうか。
　つまり位置を確定した状態では、
次にどの運動量が観測されるかという確率が全て同じ程度だということであり、
運動量についての予言能力を全く失った状態にあるのである。
　不確定性原理の一部がこんなところに顔を出している。

<P>

　「量子力学とは可算無限次元での幾何学である」と説明したかったのに、
離散的な運動量ベクトルも連続な運動量ベクトルに統合されてしまって、
結局は座標ベクトルも運動量ベクトルも連続として考えねばならぬという
全くイメージしにくい話にたどりついてしまった。
　この調子ではきっと、エネルギーや時間も連続量に違いない。
　基本的な物理量がこんな具合では、
果たして本当に可算無限次元のベクトル空間を導入する必然性はあったのだろうか？
と思えてしまう。

<P>

　まぁ、そんなに悲観することはない。
　実際にこの表現法は便利なのだし、
スピンや角運動量などは連続値をとらないのでこのイメージがそのまま使えそうではないか。
　運動量や座標が少々無茶なことをしているだけなのだと考えよう。
　それにこれはまだ次の段階へ進むための前置きに過ぎないのだ。
　そのうちすごいことが起こるのを期待しよう。
